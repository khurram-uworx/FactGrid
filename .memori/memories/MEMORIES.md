# Memories

- FactGrid architecture: .NET 10 solution with 3 projects. FactGrid (shared library with EntityRegistry, Excel parsers, template generator), FactGrid.AspNet (ASP.NET MVC host with MCP Streamable HTTP tools at /api/mcp and /api/mcp/{entityName}, IngestionController at /api/ingestion/{entityName}/upload, EntityController web UI), FactGrid.Mcp (STDIO MCP console app with DataEntryTools - generate_template, validate_excel, upload_excel, list_entities). MCP routes are AllowAnonymous. ASP.NET Identity is scaffolded but not enforced. EntityRegistry is singleton populated from FactGridEntityCatalog static catalog.
